# AvailityHW

Required to run:<br/>
Java 11.0.10<br/>
Angular CLI 11.2.7<br/>
Node 12.0<br/>
npm 6.9.0<br/>
Apache Maven 3.8.1<br/>
<br/>

-- Spring Boot --<br/>
Run on port 8080<br/>
cd /av-api<br/>
mvn clean install<br/>
java -jar target/av-api-0.0.1-SNAPSHOT.jar-<br/>
<br/>
-- Angular --<br/>
Run on port 4200<br/>
cd /av-ui<br/>
npm install<br/>
npm start<br/>
<br/>
-- H2 Database --<br/>
http://localhost:8080/h2-console/<br/>
